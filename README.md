This website is designed to be deployed directly from the root directory.

### GitHub Pages

1. **Push** the `main` branch to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **root (/)**
4. Click **Save**.

Your site will be live at `https://www.drivecityrp.net/`.
